/**
 * POST /api/admin/verify-2fa
 * Body: { code: string }
 *
 * Stage 2 of admin 2FA (Path A — WhatsApp OTP). Reads the
 * `1p1g-admin-2fa-pending` cookie set by /api/admin/login, verifies the
 * supplied 6-digit code matches the signed cookie payload, and issues

 * on the login page (which fires a fresh code).

 * customer verify-otp posture from HIGH-C1).
